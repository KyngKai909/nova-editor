# Nova Documentation

Nova is a **browser-based, Git-native visual editor** for the "last mile" of web
development. AI tools generate 90% of an app from a prompt — Nova is where you do
the final 10% by hand: click an element, tweak its layout, color, and copy, and
push clean code back to your repo. No tokens spent on micro-tweaks, no lock-in,
nothing to install.

> **Alpha software.** Nova is in active development. Expect rough edges, and
> please send feedback.

## The core loop

1. **Import** a project — a GitHub repo, a local folder, or pasted code.
2. **Edit** it visually on a live canvas (or in the built-in code editor, or with AI).
3. **Ship** it — review a clean diff and commit, push, or open a pull request.

Everything runs locally in your browser. Your files stay yours; your Git is the
source of truth.

## Who Nova is for

- **Vibe coders & solopreneurs** finishing an AI-generated site without prompt fatigue.
- **Designers** who want pixel control over a real codebase, not a proprietary canvas.
- **Developers** who want a visual layer that lives *next to* their code editor, not instead of it.
